Important FAQ

What is the Document Object Model (DOM)?

The DOM is one of the power full tools which is used to interactive the website and side by side it helps the user navigate the interface easily. Dom is a such a programming interface which is helps to build a website like a tree structure and it takes the every element as a node like tag, attributes etc. using Dom we can easily handle this element like update, added, remove.

How do you select an element from the DOM?

We can select element form DOM using 5 methods
1) getElementById()
2)getElementsByClassName()
3)getElementsByTagName()
4)querySelector()
5)querySelectorAll()

What is event delegation in the context of the DOM, and why is it useful

Event delegation is a technique where you can attach event listener over a parent element. so that when any event occurs it will reached at the child element by the parent element. If we use event delegation then we don't need use separate event listeners for multiple child elements.

How do you manipulate an element's attributes and styles using the DOM?

Using the DOM we can easily changed the HTML elements attribute and style. using setAttribute() we can change the value of attribute using style property we can changed CSS styles (like : color, fontSize)
